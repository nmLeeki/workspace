사고조사 진단 장비

  • 기능

    자동차 내의 커넥터를 통해 전자제어장치의 상태를 진단할 수 있으며 소프트웨어 업데이트, 고장 코드 기록 및 백업 등이 가능합니다.

  • 보유장비
    1. AUTEL MS909
    2. G-scan M
    3. 벤츠 진단기
    4. BMW 진단기
    5. AUDI/VW 진단기

사고기록 추출 장비

  • 기능

    사고기록장치에 기록된 자동차의 운행정보를 자동차를 통해서 혹은 사고기록장치에 직접 연결하여 추출할 수 있습니다.

  • 보유장비
    1. BOSCH CDR KIT
    2. GIT EDR KIT
    3. TESLA CDR KIT

사고재현 3D 스캐너

  • 기능

    사고현장 주변 환경의 형상, 굴곡 등의 상태를 2축 라이다와 카메라 센서를 통해 스캔하여 관련 소프트웨어를 통해 데이터를 쉽게 변환할 수 있고, 휴대가 용이합니다.

  • 보유장비
    1. LEICA BLK2GO
